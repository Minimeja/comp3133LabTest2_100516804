# COMP3133 Lab Test 2 – SpaceX App

This is an Angular application that displays SpaceX mission data using an external API.

## Live Application
https://comp3133-lab-test2-100516804-alkq.vercel.app

## GitHub Repository
https://github.com/Minimeja/comp3133LabTest2_100516804

## Features
- Displays SpaceX missions in a card layout
- Search missions by name
- Filter missions by status
- Data fetched using Angular HttpClient

## Technologies Used
- Angular
- TypeScript
- HttpClient
- CSS
